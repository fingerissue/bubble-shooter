# bubble-shooter

이 프로젝트는 고등학교 1학년, 콘솔 게임 만들기(C언어) 수행평가로 제출하였습니다.
현재는 코드를 만지작거려서 실행할 수 없는 상태가 되어버렸습니다.

이해해 주셔서 감사합니다.
